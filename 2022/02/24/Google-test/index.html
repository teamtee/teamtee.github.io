这些断言适用于窄字符串和宽字符串对象（string 和 wstring），如果在两个 C 字符串上使用，它会测试它们是否在相同的内存位置，而不是它们是否具有相同的值的。使用 EXPECT_STREQ 按值比较 C 字符串（例如 const char*）。 将指针与 NULL 进行比较时，使用 EXPECT_EQ(ptr, nullptr) 而不是 EXPECT_EQ(ptr, NULL)
